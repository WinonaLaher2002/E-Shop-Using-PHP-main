ğŸ› ï¸ Demo Setup (Local Development)
----------------------------------

To run this project locally:

1.  git clone https://github.com/WinonaLaher2002/E-Shop-Using-PHP-main.git
    
2.  **Set up your development environment**
    
    *   Install **XAMPP** or **WAMP** (includes Apache + MySQL + PHP)
        
    *   Start **Apache** and **MySQL**
        
3.  **Create the database**
    
    *   Open **phpMyAdmin**
        
    *   Create a new database (e.g., eshop\_db)
        
    *   Import the SQL file from the sql/ folder to create tables
        
4.  **Configure the database connection**
    
    *   Edit your database config file (in inc/, e.g., config.php)
        
    *   Update DB host, username, password, and database name
        
5.  **Run the app**
    
    *   Place the project in your server root (e.g., htdocs/eshop)
        
    *   Open in browser:[http://localhost/eshop/](http://www.eshopfatimanianstechnocrats.42web.io/)
